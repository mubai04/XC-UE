# 00 跨层契约说明

版本：S2A 冻结候选（v2）  
状态：契约定义完成；**运行时尚未迁移**

## 1. 目的

冻结 L0→L3 各层对象边界、字段语义、状态机与引用规则，消除同名异义字段（`主失败`、`失败类型`、`修复方向`、`status` 等）。

## 2. 四层产物

| 层级 | 产物对象 | v2 Schema |
|------|----------|-----------|
| L1 | 发现项、失败包 | `L1发现项结构_v2.json`、`L1失败包结构_v2.json` |
| L1.5 | 路由决策 | `L1_5路由决策结构_v2.json` |
| L2 | 修复单、报告 | `L2修复单结构_v2.json`、`L2报告结构_v2.json` |
| L3 | 执行任务包、执行结果 | `L3执行任务包结构_v2.json`、`L3执行结果结构_v2.json` |

公共引用：`公共引用结构_v1.json`

## 3. 引用原则

- 下游通过**对象编号**引用上游，不得复制上游对象后重新解释字段。
- 可携带**显示摘要**，但必须标注为非权威。
- 禁止无层级限定的 `status`、`最终状态`、`处理状态`。

## 4. 运行时状态（S2A）

- v1 Schema 仍为生产运行契约。
- v2 为冻结候选，S2A **不**同时写入 v1/v2。
- S2B 迁移须显式边界 + 迁移器，禁止静默 fallback。
