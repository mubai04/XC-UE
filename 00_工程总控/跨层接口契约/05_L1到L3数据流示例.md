# 05 L1 到 L3 数据流示例

同一问题从头到尾的编号引用链（虚构案例，非 pilot 语料，非标准答案）。

## 场景

章节正文出现**局部句式重复、信息增量不足**，L1 文风域发现可路由信号，经 L1.5 路由至 L2-02，L2 产出修复单，L3 生成候选补丁。

## 引用链摘要

```text
L1F-DEMO-001 → L1P-DEMO-001 → L15R-DEMO-001 → L2F-DEMO-001 → L2R-DEMO-001 → L3T-DEMO-001 → L3E-DEMO-001
```

每一层仅携带上游**编号引用** + 本层专有字段，不复制重解释上游对象。
