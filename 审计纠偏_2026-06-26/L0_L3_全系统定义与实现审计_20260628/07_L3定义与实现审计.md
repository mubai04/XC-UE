## 1. L3 是规划层还是执行层

| 来源 | 声明 |
|------|------|
| L3-00 | L1/L1.5/L2 → L3 任务 → 文件/日志 → L1/L1.5 复验 |
| L3-02 | 候选正文生成/改写；**不**做质量判断 |
| L3-99 | 禁止覆盖正式章节、禁止质量裁决 |
| `protocol_rules.json` | `formal_prose_modification: "否"` |
| 代码 | 默认 `TASK_PLANNING_ONLY`；可选 `CANDIDATE_GENERATION` |

**判定：** L3 = **任务规划层为主** + **受限候选执行**（`_candidates/`  only）。
