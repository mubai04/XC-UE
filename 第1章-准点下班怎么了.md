# 第1章 准点下班怎么了

> 这本书不解决任何问题。它只让一个人替你准点下班。

下午六点整。陈敛关掉IDE，拿起桌上的工牌，站起来。

旁边的工位，小刘正在以肉眼可见的速度衰老。屏幕开着六个窗口，桌上摆着三杯已经凉透的咖啡，眼神介于"还能再战"和"我觉得我需要去医院"之间。

陈敛瞥了一眼小刘的手背——上面有一道没擦干净的血痕，从虎口延到手腕。他记得小刘今天没切过任何东西。但他没问。不问比较省事。

"走了？"小刘抬头，语气里带着一丝不可思议。

"走了。"陈敛把卫衣帽子戴上，"你也早点。"

"我把这个bug修完。"

陈敛看了一眼他的屏幕——那个bug小刘从下午两点开始修到现在。以陈敛的经验，最多二十分钟能解决。但他不打算说。

原因有两个。第一，他不想暴露自己其实懂。第二——小刘修得越久，系统积分涨得越多。

电梯门关上。他的眼前弹出一个半透明的面板。

```
【今日收割简报】
今日摸鱼时长：7小时48分钟
今日收割积分：+1,247
本月累计：+16,234
收割分布：单一来源占比偏高（邻座：37%）
建议：分散收割源以降低暴露风险。
```

陈敛看了看那行"降低暴露风险"，心里默默给系统加了一分——还挺专业。但他没打算改，反正也没人发现。

电梯下行。他看着面板上那个积分数字，想起了三个月前的那个晚上——躺在床上刷手机，眼前突然弹出这块面板。他关了三次，它弹了四次。第四次他把被子蒙在头上说了一句"明天再说"，然后面板回了他四个字。

**"好的。晚安。"**

后来他花了一周才搞明白运作逻辑——摸鱼就涨积分，别人加班他也在涨，他什么都不用做。搞明白之后他坐在工位上沉默了五分钟。表情介于"还有这种好事"和"那我这三个月岂不是白活了"之间。

然后他打开了视频网站。

反正系统已经自动跑了三个月，积分都攒到五位数了。

电梯到了一楼。门打开。

他走出大楼，夕阳落在对面写字楼的玻璃幕墙上，反射出一片金色的光。手机震动。部门群消息——赵总监：

"@所有人 今晚加个班，下周的项目方案要赶出来。食堂有晚餐。"

群里立刻弹出一片"收到""好的赵总""没问题"。

陈敛打了三个字："家有急事。"发完把手机揣回兜里，走向新城轨道站。

---

第二天上午，陈敛准时出现在工位上。

不是他不想迟到——是系统的"反检测屏障"需要他人在大楼范围内才能全功率运行。他试过请假在家，那天的积分只有平时的三分之一。从那以后他每天都来。不是为了上班，是为了"信号满格"。

十一点半。午休时间。

陈敛把椅背调到最大角度，卫衣帽子往脸上一拉，开始睡觉。旁边的键盘声密集得像机关枪——小刘在赶一份没人催的方案，敲得眼冒金星。

陈敛睡得很安稳。

两点整。他醒了，伸了个懒腰。系统面板自动弹出：

```
午休时段收割积分：+623
主要来源：邻座（+412）
次要来源：同楼层未知目标（+211）
```

他打了个哈欠，关掉面板。不错。睡个觉又收了六百多。

他拿起水杯去接水。茶水间的走廊拐角，他差点跟一个人撞上——前台林栀，手里拿着一个快递信封。

"啊，抱歉。"她笑了一下，"你是技术部的……陈敛？"

"嗯。"

"有你的快递。我正想送上去。"她把信封递给他。

陈敛接过来看了一眼——发件人空白，寄件地址空白。他翻了翻，里面是一张纸。

他没当着她的面拆。点头说了声谢，走回工位。

坐下来之后他拆开信封，里面是一张A4纸，上面只有一行打印体字：

**"你的工位位置，是三十二楼最后一个靠窗位。你知道上一个坐这里的人去哪了吗？"**

没有署名。没有日期。没有寄件人。

陈敛盯着那行字看了十秒。

"上一个坐这里的人，"他自言自语，"关我什么事。"

他把纸折好塞回信封，扔进抽屉里。

但他又看了那个抽屉一眼。

他拿出手机，想给系统打字："你知道这是谁送的吗？"

字还没打完，他删了。他不想让系统知道他有点在意这件事。

下午三点。赵总监的纸条来了——"小陈，到我办公室来一趟。"

陈敛把纸条揉成一团，起身。

赵总监办公室在三十二楼尽头，落地窗，视野很好。陈敛进去的时候，赵总监正坐在办公桌后面，表情介于"我要跟你谈谈"和"其实我也没什么好说的"之间。

"坐。"

陈敛坐下。

"小陈啊，你来公司三个月了。"

"嗯。"

"你的工作……"赵总监斟酌了一下用词，"还可以。但你有没有发现，你是全部门下班最早的？"

"我活干完了。"

赵总监愣了一下。这句话的潜台词太明确了——"我活干完了，不加班有什么问题？"但作为总监，他不能接这个话，因为接了就等于承认"不加班是对的"。

"我不是说你不能下班。我是说——年轻人嘛，多花点时间在工作上，对你以后的晋升有好处。"

陈敛点了点头："好的赵总，我记住了。"

他站起来，走出办公室。

回工位的路上他路过赵总监桌上的台历——翻到六月那页。他无意间扫了一眼，某一个日期上画着一个很轻的铅笔圈。不是会议标注，不是截止日。就是一个圈。

他没多想。回了自己工位，打开电脑，切到视频网站，戴上耳机。

工位旁边的亚克力立牌："代码调试中——请勿打扰。"

赵总监从办公室门口路过，看到陈敛在认真敲代码，满意地点了点头。

他不知道陈敛的屏幕上开着一个隐藏窗口——番剧第三集，进度条走了一半。刚才敲的那几行代码是他两天前就写好的，现在只是复制粘贴。快捷键按得飞起。

下午五点半。系统弹了一条消息。

```
【预估：今日总收割量可达 2,000+ 积分】
主要得益于部门集体加班氛围。
建议宿主保持当前状态。
```

陈敛打字回复："能不能不要每次弹窗都带建议？我知道该怎么做。"

系统沉默了大约三秒——

```
【已记录。】
【下次减少建议频率。】
```

陈敛觉得这个系统有时候还挺好说话的。

下午六点整。关电脑，拿工牌，站起来。

小刘不在工位上——洗手间的方向传来水龙头开到最大的声音。陈敛看了一眼他的桌面：键盘上搁着一板止痛药，锡纸板被抠了两个洞。

他停顿了一秒。然后走了。

走出大楼的时候，晚风迎面吹过来。他在台阶上站了两秒，然后做了一个他自己也没想到的动作——他回头，看了一眼身后的大楼。

三十二层的全玻璃幕墙在黄昏的天色里反射出一层暗金色的光。大楼里灯火通明，至少还有一半楼层亮着。他之前从来没仔细看过这栋楼的轮廓。现在认真地看了一眼——他发现了一个所有人都忽略了的事情。

这栋楼的横截面，不是一个正方形。

是一个不规则的**八角形**。

他站在那看了三秒。正要转身走，手机在口袋里震了一下——不是来电，是系统推送。

```
【补充数据】
这座大楼的建筑设计图，在规划局没有备案。
一次都没有。
```

陈敛的脚步顿了一下。

然后他继续走了。

"关我什么事。我又不是规划局的。"

他把手机放回口袋，继续走。

走了十几步，又停下来。他掏出手机——翻到那条推送，截了个图。等反应过来的时候，截图已经存好了。

他看了一眼屏幕。把手机塞回口袋。

"手快了。"
