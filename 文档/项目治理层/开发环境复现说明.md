# 开发环境复现说明

## 平台

- **操作系统**：Windows 10/11
- **Python**：3.11 及以上（干净环境验证使用 **3.13.13**）

## 依赖安装

在仓库根目录执行：

```powershell
python -m venv .venv
.\.venv\Scripts\python.exe -m pip install -U pip
.\.venv\Scripts\python.exe -m pip install -e ".[dev,runtime]"
```

或使用锁定文件复现（需先创建虚拟环境并安装 editable 包）：

```powershell
.\.venv\Scripts\python.exe -m pip install -r "依赖锁定_开发运行.txt"
.\.venv\Scripts\python.exe -m pip install -e ".[dev,runtime]"
```

锁定文件路径：`依赖锁定_开发运行.txt`（由干净临时环境 `pip freeze --exclude-editable` 生成，不含 API Key 与 editable 路径）。

## 受控依赖范围（pyproject.toml）

| 包 | 范围 |
|----|------|
| jsonschema | >=4.23.0,<5 |
| referencing | >=0.35.0,<1 |
| PyYAML | >=6.0.2,<7 |
| pytest | >=8.0,<9 |

干净环境实测版本（2026-06-28）：jsonschema **4.26.0**，referencing **0.37.0**。

## Schema 验证

```powershell
python "脚本/校验_L0至L3跨层接口契约.py"
```

预期输出包含：

```text
SCHEMA_CHECK = PASSED
REFERENCE_RESOLUTION = PASSED
POSITIVE_FIXTURES = PASSED
NEGATIVE_FIXTURES = PASSED
VALIDATION_OK
```

离线迁移预检：

```powershell
python "脚本/回放_v1到v2跨层迁移.py" --validate-only
```


## 说明

- v2 Schema 使用绝对 ASCII `$id`（`xcue://schemas/cross-layer/...`）与 ASCII `$anchor` 引用，不依赖网络或文件名 `$ref`。
- 生产运行时仍为 **v1**；S2B-2 运行时切换尚未启动。
