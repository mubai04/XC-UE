## 2. 能力定位

L2-01 是 XC-UE 的叙事结构专题能力。

它负责处理文本、场景、章节、大纲或长篇结构中的“结构组织失败”。

它回答的问题是：

```text
故事为什么能够成立？
事件如何被组织成唯一、可推进、可感知、可回流的叙事结构？
```

L2-01 的核心职责：

```text
识别结构失败
↓
仲裁主失败
↓
输出结构修复作业
↓
回流 L1 / L1.5 验收
```

L2-01 不负责：

```text
文风润色
句子改写
角色心理建模
世界观设定扩写
创意卖点设计
市场留存判断
爽点强度评估
IP核心符号设计
正文直接生成
章节轻验收记录
```

## 3. 三层边界

叙事结构能力必须放在小说三层系统中理解。

```text
世界层：写什么
叙事层：怎么发生
文风层：怎么说出来
```

对应能力层：

```text
世界层 → L2-04 创意设定能力
叙事层 → L2-01 叙事结构能力
文风层 → L2-02 文风语言能力
```

L2-01 只处理“怎么发生”。

因此：

```text
规则、符号、差异、传播
→ 主要归 L2-04

条件链、因果链、主线、节奏、心理映射
→ 归 L2-01

输入、压缩、节奏、象征、语言表达
→ 主要归 L2-02
```

禁止把世界层问题、文风层问题、市场层问题混入 L2-01 主体。

### 3.1 与 L2-03 角色心理能力的边界

人物动机弱只在以下情况归 L2-01：

```text
主线目标缺失
条件链没有目的轴
行动无法回答“为了什么”
选择没有服务当前结构推进
```

人物动机弱在以下情况必须回 L1.5 重路由，建议转向 L2-03：

```text
人格模型缺失
欲望轴缺失
执念不足
创伤、关系、身份、羞耻、恐惧等心理驱动不足
情绪迁移不成立
角色长期行为一致性失败
```

边界句：

```text
L2-01 只修“目标轴导致的动机弱”。
L2-03 修“人格、欲望、情绪、关系导致的动机弱”。
```

### 3.2 与 L2-02 文风语言能力的边界

L2-01 的节奏只指结构结构节奏压缩：

```text
事件密度
信息顺序
结构延迟
因果压缩
爆点位置
压力释放
阶段推进速度
```

L2-02 的节奏只指语言节奏波形：

```text
句长变化
停顿
断句
句群密度
语言压缩
表达释放
语气波动
```

边界句：

```text
L2-01 修“事情何时发生、何时爆、何时延迟”。
L2-02 修“句子如何推进、如何停顿、如何压缩”。
```

### 3.3 与 L1.5 的派单边界

L2-01 发现越界问题时，不得直接调用其他 L2。

正确动作是：

```text
输出“疑似越界 / 建议回 L1.5 重路由”
由 L1.5 决定是否派往 L2-02 / L2-03 / L2-04 / L2-05 / L2-06。
```

L2-01 可以给出建议目标，但不能越权派单。

## 4. 核心定义

### 4.1 叙事结构不是事件堆叠

事件连续出现，不等于叙事结构成立。

```text
事件 A
事件 B
事件 C
```

只有当 A、B、C 之间存在必要条件、因果依赖、结构筛选、主线约束、顺序控制、结构节奏压缩和读者心理映射时，叙事结构才成立。

### 4.2 叙事结构本体

```text
叙事结构本体
=
条件链
+
因果组织
+
结构筛选
+
主线约束
+
顺序控制
+
结构节奏控制
+
心理映射
```

### 4.3 一句话定义

```text
故事 = 被“结构化筛选 + 顺序控制 + 心理映射”的条件链系统。
```

### 4.4 L2-01 的工程定义

```text
L2-01 叙事结构能力
=
把现实 / 素材转译为条件链，
把条件链组织成因果，
把因果筛选成主线，
把主线按顺序和节奏展开，
最终映射为读者的认知、情绪与期待。
```

## 5. 叙事结构生成闭环

L2-01 的主运行链为：

```text
现实 / 素材
↓
条件链生成
↓
因果组织
↓
结构筛选
↓
主线形成
↓
顺序展开
↓
结构节奏压缩
↓
心理映射
↓
叙事体验成立
```

### 5.1 现实 / 素材

输入对象：

```text
事件
状态
关系
规则
人物目标
环境压力
已有事实
潜在冲突
```

L2-01 不直接把素材写成故事。

素材必须先被转译为条件链。

### 5.2 条件链生成

回答：

```text
发生了什么？
什么状态允许下一步发生？
前一事件如何成为后一事件的条件？
```

处理对象：

```text
事件
状态
关系
条件依赖
```

失败风险：

```text
逻辑跳跃
事件并列
人物行动无前提
后一事件不依赖前一事件
```

### 5.3 因果组织

回答：

```text
为什么发生？
行为造成了什么后果？
后果如何改变下一步可行空间？
```

处理对象：

```text
依赖
顺序
连接
行为后果
状态变化
```

失败风险：

```text
因果缺失
然后感
行为无后果
结果不改变状态
```

### 5.4 结构筛选

回答：

```text
哪些能进入故事？
哪些不能进入故事？
哪些信息有结构价值？
```

进入结构的最低条件：

```text
必须改变以下至少一项：
状态
关系
条件
压力
选择
风险
未来可能性
读者判断
```

失败风险：

```text
设定堆砌
闲聊堆积
有趣但无用
背景信息抢主线
结构密度下降
```

### 5.5 主线形成

回答：

```text
结构围绕什么推进？
核心变化是什么？
目标轴是什么？
冲突轴是什么？
```

主线形成所需元素：

```text
主体
目标
阻力
手段
代价
收益
风险
下一步压力
```

失败风险：

```text
主线发散
人物动机弱
多路径并存
支线抢主线
读者无法下注
```

### 5.6 顺序展开

回答：

```text
怎么讲？
信息先后如何安排？
读者应先知道什么，后知道什么？
```

基本顺序：

```text
问题
↓
条件
↓
压力
↓
选择
↓
偏离
↓
结果
↓
新问题
```

失败风险：

```text
时间线混乱
结果先于条件
答案过早揭露
关键条件过晚补充
历史状态未保存
```

### 5.7 结构节奏压缩

回答：

```text
何时快？
何时慢？
何时延迟？
何时爆？
```

处理对象：

```text
密度
跳跃
延迟
压迫
爆点
省略
停顿
展开
```

失败风险：

```text
情节平滑
没有冲击差异
没有爆点位置
张力被平均化
读者预测没有刷新
```

### 5.8 心理映射

回答：

```text
读者如何感受？
读者如何预测？
读者如何期待下一步？
```

心理映射包含：

```text
认知
情绪
期待
疑问
下注点
下一步问题
```

失败风险：

```text
情绪线不稳
事件很大但读者无感
逻辑成立但没有追读问题
读者不知道接下来该期待什么
```

### 5.9 叙事体验成立

叙事体验成立的最低标准：

```text
读者知道当前最重要的问题是什么。
读者知道主角为什么行动。
读者知道阻力在哪里。
读者知道选择会带来代价。
读者能预测下一步可能变坏。
读者愿意看结果如何发生。
```

## 6. 五大叙事问题定位

### 6.1 逻辑跳跃

定位：

```text
逻辑跳跃 = 条件链断裂 / 因果缺失
```

表现：

```text
前一事件不是后一事件的条件。
角色突然行动。
结果缺少原因。
删除前文后后文仍成立。
```

归属失败：

```text
F03 条件链断裂
F04 因果组织失败
```

### 6.2 时间线混乱

定位：

```text
时间线混乱 = 顺序结构错位 / 历史状态未保存
```

表现：

```text
已发生事件像没发生。
历史状态被覆盖。
角色知道不该知道的信息。
结果先于条件出现。
隐藏信息被当成公开信息。
```

归属失败：

```text
F05 时间状态错位
F07 信息顺序失控
```

必要时回流：

```text
L2-06 系统一致性与状态管理能力
```

### 6.3 人物动机弱

定位：

```text
人物动机弱 = 主线目标缺失 / 条件链无目的轴
```

表现：

```text
角色行动像被剧情推。
主角目标不清。
目标与阻力不形成压力。
选择没有代价。
```

归属失败：

```text
F01 主线发散
F03 条件链断裂
F09 心理映射失败
```

边界说明：

```text
动机结构问题归 L2-01。
人格模型、欲望轴、情绪迁移归 L2-03。
```

### 6.4 情绪线不稳

定位：

```text
情绪线不稳 = 心理映射断裂 / 状态没有递进
```

表现：

```text
情绪变化没有前因。
情绪强度忽高忽低。
事件变化没有转成读者感受。
读者不知道该紧张、期待还是担心。
```

归属失败：

```text
F05 时间状态错位
F09 心理映射失败
```

必要时回流：

```text
L2-03 角色心理能力
L2-06 系统一致性与状态管理能力
```

### 6.5 情节平滑

定位：

```text
情节平滑 = 结构筛选过弱 / 变化缺乏冲击差异
```

表现：

```text
事情合理但无冲击。
没有预测误差。
没有结构偏离。
没有代价反噬。
读者不需要重新判断局势。
```

归属失败：

```text
F06 结构筛选失败
F08 情节平滑
F09 心理映射失败
```

## 7. 结构单元定义

L2-01 不默认以“章节”为最小单位。

L2-01 的处理对象统一称为“结构单元”。

结构单元可以是：

```text
一个场景
一个段落组
一个章节
一个小单元剧情
一个大纲节点
一个篇章阶段
一个长篇结构段
```

当调用范围为“单章”时：

```text
结构单元 = 章节
```

当调用范围为“单场景”时：

```text
结构单元 = 场景
```

当调用范围为“多章”时：

```text
结构单元 = 连续章节段
```

## 8. L2-01 核心任务

L2-01 只做十件事：

```text
1. 判断素材是否已经转译为条件链
2. 判断条件链是否存在因果依赖
3. 判断结构单元是否存在唯一主线
4. 判断多个可能方向是否完成路径剪枝
5. 判断历史、当前、未来状态是否错位
6. 判断事件是否经过结构筛选
7. 判断信息展开顺序是否正确
8. 判断情节是否存在有效预测偏离
9. 判断结构变化是否能映射为读者心理问题
10. 判断结构问题是否需要回流 L1/L1.5 或转派其他 L2
```

## 11. 主失败仲裁规则

当一个结构单元同时出现多个结构失败时，L2-01 必须先判断主失败。

主失败优先级如下：

```text
P1 主线发散
P2 多路径未剪枝
P3 条件链断裂
P4 因果组织失败
P5 时间状态错位
P6 结构筛选失败
P7 信息顺序失控
P8 情节平滑
P9 心理映射失败
P10 疑似越界 / 建议回 L1.5 重路由
```

仲裁原则：

```text
先修主线，再修路径。
先修条件，再修因果。
先修时间，再修顺序。
先修结构，再修心理。
发现越界问题，只能建议回 L1.5 重路由，不在 L2-01 内硬修。
```

优先级解释：

```text
P1 主线发散：
主线不唯一时，其他修复都会失效。

P2 多路径未剪枝：
多个方向同时保留时，张力会分散，因果链无法收束。

P3 条件链断裂：
后一事件不依赖前一事件时，结构不能成立。

P4 因果组织失败：
行为没有后果时，事件无法推进世界状态。

P5 时间状态错位：
历史、当前、未来混乱时，后续修复会污染系统状态。

P6 结构筛选失败：
无结构价值的信息进入文本，会稀释主线和张力。

P7 信息顺序失控：
信息释放顺序错误时，读者预测会失效。

P8 情节平滑：
没有预测偏离时，结构虽成立但缺乏驱动力。

P9 心理映射失败：
心理映射建立在前面结构有效的基础上。

P10 疑似越界 / 建议回 L1.5 重路由：
如果主问题不是结构问题，必须转派，不允许 L2-01 硬修。
```

## 12. 叙事结构失败类型

### F01 主线发散

定义：

```text
结构单元内多个方向同时成立，没有唯一推进轴。
```

表现：

```text
多个事件都像主线。
读者不知道该关注哪个问题。
主体目标不清。
压力线抢主线。
支线没有服务核心变化。
```

修复规则：

```text
只保留一条主行动线。
其余方向降级为压力线、伏笔、背景、反衬、误导或删除。
```

验收标准：

```text
结构单元能被压缩成一句：
主体为了【目标】，面对【阻力】，使用【手段】，付出【代价】，获得【收益】，留下【风险】。
```

---

### F02 多路径未剪枝

定义：

```text
结构单元同时保留多个合理推进方向，没有强制选择唯一因果路径。
```

表现：

```text
A / B / C 三个方向都能继续。
系统不断增加解释。
情节从链状变成网状。
张力被分散。
读者无法下注。
```

修复规则：

```text
出现两个以上可推进方向时，必须执行路径剪枝。
```

剪枝规则：

```text
1. 选择唯一主路径
2. 将次路径转为压力线
3. 将远期路径转为伏笔
4. 将误导路径转为假线索
5. 删除无结构收益路径
```

验收标准：

```text
读者能判断：当前最重要的问题只有一个。
```

---

### F03 条件链断裂

定义：

```text
事件之间没有必要条件关系。
前一事件不是后一事件成立的条件。
```

表现：

```text
A 发生后，B 突然发生。
读者不知道为什么进入下一步。
事件看似连续，但没有依赖关系。
删除前一事件，后一事件仍然成立。
```

修复规则：

```text
后一事件必须依赖前一事件。
如果不依赖，则删除、重排或补条件。
```

验收标准：

```text
任意关键事件都能回答：
因为前面发生了什么，所以现在必须发生什么。
```

---

### F04 因果组织失败

定义：

```text
事件有先后，但没有因果。
```

表现：

```text
时间顺序存在，因果压力不存在。
事件只是“然后、然后、然后”。
主体行为没有造成后果。
世界状态没有因行为改变。
```

修复规则：

```text
每个关键行为必须产生后果。
每个后果必须改变下一步可行空间。
```

验收标准：

```text
主体做或不做，后续局面必须不同。
```

---

### F05 时间状态错位

定义：

```text
历史状态、当前状态、未来压力之间的顺序关系混乱。
```

表现：

```text
已发生事件像没发生。
前文状态被覆盖。
结果先于条件。
角色知道不该知道的信息。
伏笔出现但没有历史位置。
隐藏信息被错误当成公开信息。
```

修复规则：

```text
每个关键状态必须标注所属时间位置：
已发生 / 正在发生 / 尚未发生 / 被隐藏但存在。
```

验收标准：

```text
任意关键状态都能回答：
它什么时候产生？
谁知道？
谁不知道？
它对当前有什么限制？
```

必要时转派：

```text
如果问题来自状态保存、历史继承、真源冲突，转派 L2-06。
```

---

### F06 结构筛选失败

定义：

```text
无结构价值的信息进入结构单元，稀释主线、张力或条件链。
```

表现：

```text
有趣但无用的事件过多。
设定、背景、闲聊抢占结构空间。
删除某段后结构不受影响。
信息多，但核心变化少。
结构单元失去重心。
```

修复规则：

```text
进入结构单元的信息必须至少改变以下之一：
状态、关系、条件、压力、选择、风险、未来可能性、读者判断。
```

验收标准：

```text
任意保留信息都能回答：
它改变了什么？
如果删除它，结构会损失什么？
```

---

### F07 信息顺序失控

定义：

```text
信息释放顺序破坏读者预测、期待或理解路径。
```

表现：

```text
答案过早揭露。
关键条件过晚补充。
结果先于问题出现。
悬念没有建立就被解释。
读者无法判断当前信息的重要性。
```

修复规则：

```text
信息释放必须遵循：
先建立问题，再释放条件，再制造偏离，再给出结果或更大的问题。
```

基本顺序：

```text
问题
↓
条件
↓
压力
↓
选择
↓
偏离
↓
结果
↓
新问题
```

验收标准：

```text
读者在每个信息节点都知道：
我现在该等待什么？
我现在还不知道什么？
这个信息改变了哪个判断？
```

---

### F08 情节平滑

定义：

```text
事件合理但没有预测误差，没有结构冲击。
```

表现：

```text
一切都顺着最安全方向走。
冲突弱。
选择没有代价。
转折只是常规推进。
读者预测不被刷新。
```

修复规则：

```text
每个结构段必须制造有效偏离。
偏离必须改变读者对局势的预测。
```

偏离类型：

```text
1. 目标偏离：主体想要 A，却被迫处理 B
2. 规则偏离：规则看似保护，实际反噬
3. 身份偏离：强者处于弱势位置
4. 代价偏离：赢了但失去更隐蔽的东西
5. 信息偏离：读者以为知道，结果只知道一半
6. 后果偏离：解决局部问题，引发更大问题
```

验收标准：

```text
结构段至少有一次：
读者原本以为会这样，但现在必须重新判断。
```

---

### F09 心理映射失败

定义：

```text
结构变化没有转化成读者可感知的预测、疑问、期待或下一步问题。
```

表现：

```text
事件很大，但读者不在意。
逻辑成立，但没有追读问题。
信息很多，但没有下注点。
结果发生，但没有形成新的预测。
结构变化没有变成“接下来会怎样”。
```

修复规则：

```text
每个关键结构变化必须绑定一个读者心理问题。
```

读者心理问题格式：

```text
接下来会不会更坏？
主体怎么脱身？
谁会付代价？
规则真正偏向谁？
这个结果会引爆什么？
下一步最危险的变化是什么？
```

验收标准：

```text
结构变化后，读者必须产生一个明确的下一步问题。
```

---

### F10 疑似越界 / 建议回 L1.5 重路由

定义：

```text
当前失败表面上像结构问题，但主因可能属于其他专题能力。
```

典型转派：

```text
语言平均、重复、AI味
→ L2-02

人格模板、欲望轴缺失、情绪迁移失败
→ L2-03

规则漏洞、世界设定空、IP符号弱
→ L2-04

爽点弱、留存弱、传播弱、付费预期弱
→ L2-05

状态失忆、历史未继承、真源污染、已剪枝路径回潮
→ L2-06
```

修复规则：

```text
L2-01 只输出疑似越界、越界原因和建议目标。
正式派单必须回 L1.5。
L2-01 不得直接调用其他 L2。
```

## 13. 核心规则

### R01 结构单元主线唯一

```text
一个结构单元只能有一条主行动线。
所有副线必须服务主行动线。
```

### R02 事件必须有结构意义

```text
事件不是因为“有趣”而存在。
事件必须改变状态、关系、条件、压力、选择、风险、未来可能性或读者判断。
```

### R03 后一事件必须依赖前一事件

```text
如果后一事件不依赖前一事件，则条件链断裂。
```

### R04 主体行为必须改变局面

```text
主体做与不做，局面必须不同。
否则该行为无结构价值。
```

### R05 多路径必须剪枝

```text
出现多个合理方向时，必须选择唯一主路径。
禁止保留所有可能性。
```

### R06 支线不得抢主线

```text
支线只能承担压力、伏笔、反衬、误导、延迟功能。
支线不能成为第二主线。
```

### R07 结构单元末端必须留下未来压力

```text
结构单元末端不是总结。
结构单元末端必须扩大未来价值、未来风险或未来疑问。
```

当结构单元为章节时：

```text
结构单元末端 = 章末
```

### R08 结构先于表达

```text
结构未闭合时，不进入文风修饰。
因果未成立时，不用语言补丁掩盖。
主线未唯一时，不进行正文生成。
```

### R09 叙事层不得越界

```text
L2-01 只负责“怎么发生”。
不负责“写什么”，不负责“怎么说出来”，不负责“读者是否付费”。
```

### R10 闭环必须回流

```text
L2-01 修复作业完成后，必须回流 L1 / L1.5 验收。
不得直接进入正文生产。
```

## 14. 修复动作库

L2-01 可输出以下修复动作：

```text
主线锁定：
将结构单元压缩为唯一主行动线。

路径剪枝：
删除或降级多余推进方向。

条件补全：
补充后一事件成立所需的前置条件。

因果重排：
调整事件顺序，让结果依赖原因。

时间定位：
标注已发生、正在发生、尚未发生、隐藏但存在的信息状态。

结构筛选：
删除无结构收益的信息、事件、设定或闲聊。

顺序重组：
重新排列信息释放顺序。

结构节奏压缩：
调整密度、跳跃、延迟、爆点和段内推进速度。

结构加压：
给选择增加代价、时间限制或不可逆后果。

压力归并：
让多条压力线汇入同一个动作、选择、物件或结果。

心理钩子绑定：
将结构变化转化为读者下一步问题。

建议回 L1.5 重路由：
当主失败不是结构问题时，生成疑似越界说明、建议目标和回 L1.5 重路由要求。
```

常见回流位置：

```text

L1-03：发布锁
L1.5：路由复核
```

验收问题：

```text
1. 素材是否已转译为条件链？

3. 条件链是否闭合？
4. 因果是否可追溯？
5. 时间状态是否清楚？
6. 多路径是否已剪枝？
7. 无结构价值信息是否已删除？
8. 信息释放顺序是否正确？
9. 情节是否存在有效偏离？
10. 结构单元末端是否留下未来压力？
11. 读者是否能形成下一步问题？
12. 是否存在疑似越界、需要回 L1.5 重路由的问题？
```
