```text
识别创意设定失败
```

## 3. L2-04 的根部结构

### 3.1 三大表象问题

L2-04 的表层问题不是“设定不够多”，而是：

```text
1. 世界观 / 规则不统一
2. IP核心符号弱化
3. 创新受限
```

对应表现：

```text
世界观 / 规则不统一：
设定自相矛盾，规则前后冲突，逻辑断裂。

IP核心符号弱化：
记不住，传不开，缺乏识别度与传播力。

创新受限：
套路化，缺乏新意，难以产生惊喜与冲击。
```

### 3.2 三大根本原因

表象背后的根本原因是：

```text
1. 约束不统一
2. 缺乏有效压缩
3. 缺乏有效偏离
```

解释：

```text
约束不统一：
规则或者设定没有稳定的约束体系。

缺乏有效压缩：
无法将复杂世界压缩为可记忆、可传播、可进入的符号节点。

缺乏有效偏离：
没有基于规律的关系重组，预期无法被击穿。
```

### 3.3 三大深层本质

```text
约束 = 定义边界
压缩 = 建立入口
偏离 = 打破预期
```

进一步解释：

```text
约束决定：
什么能发生，什么不能发生。
边界内才有秩序与代价。

压缩决定：
读者如何进入这个世界。
复杂信息必须被压缩成可记忆入口。

偏离决定：
读者为什么觉得新。
在边界内制造认知偏离，产生新鲜感与创新体验。
```

### 3.4 统一本质

约束、压缩、偏离的共同作用是：

```text
构建可运行认知模型
```

世界观不是作者脑中的设定堆。

世界观应该是：

```text
读者脑中可以运行的模型。
```

可运行认知模型必须具备三大特征：

```text
1. 可预测
   有规律，读者能推演后果。

2. 可参与
   有关联，读者能代入选择。

3. 可验证
   有反馈，读者能通过事件确认规则有效。
```

### 3.5 世界感生成链

为什么有的模型会成为“世界”？

```text
规律运行
↓
反复验证
↓
可预测性
↓
可参与性
↓
价值产生
↓
意义形成
↓
内化沉浸
```

世界感的公式：

```text
世界感 = 相信 → 预测 → 参与 → 模拟 → 内化
```

### 3.6 真正驱动力

真正的驱动力不是设定数量。

真正的驱动力是：

```text
围绕“中心价值”运行的系统。
```

中心价值的作用：

```text
提供世界重心
统一所有设定与规则
决定冲突与故事动力
强化识别与记忆锚点
```

中心价值运行链：

```text
规则
↓
产生后果
↓
影响角色
↓
改变状态
↓
价值波动
↓
推动选择
↓
驱动故事
```

### 3.7 终极根问

L2-04 的终极问题不是：

```text
如何想一个新设定？
```

而是：

```text
如何找到一个足以统治整个世界的“中心价值”？
```

## 4. 三层边界

创意设定能力必须放在小说三层系统中理解。

```text
世界层：写什么
叙事层：怎么发生
文风层：怎么说出来
```

对应能力层：

```text
世界层 → L2-04 创意设定能力
叙事层 → L2-01 叙事结构能力
文风层 → L2-02 文风语言能力
```

L2-04 只处理“写什么世界”。

因此：

```text
中心价值、约束边界、压缩入口、有效偏离、可运行认知模型、世界后果、世界/规则符号、可持续玩法
→ 归 L2-04

条件链、因果链、主线、结构节奏、心理映射
→ 主要归 L2-01

语言表达、语境锁定、语言节奏波形、AI味抑制
→ 主要归 L2-02
```

## 5. 与其他 L2 的边界

### 5.1 与 L2-01 叙事结构能力的边界

L2-04 负责规则和世界是否能制造可写空间：

```text
规则是否有边界
规则是否制造选择
规则是否带来代价
世界是否可运行
中心价值是否驱动冲突
```

L2-01 负责这些规则进入剧情后如何组织成事件：

```text
条件链
因果组织
主线形成
顺序展开
结构节奏压缩
心理映射
```

边界句：

```text
世界规则能不能制造选择 → L2-04。
选择如何变成事件链 → L2-01。
```

### 5.2 与 L2-02 文风语言能力的边界

L2-04 负责世界 / 规则符号系统：

```text
世界观核心符号
规则符号
制度符号
禁忌符号
可传播识别点
记忆锚点
```

L2-02 负责局部语言符号触发：

```text
句子记忆点
段内物件
局部意象
场景表达锚点
```

边界句：

```text
符号能否代表世界和规则 → L2-04。
符号如何在句子和段落中被说出来 → L2-02。
```

### 5.3 与 L2-03 角色心理能力的边界

L2-04 负责世界对人的限制和诱惑：

```text
世界如何改变身份、资源、关系、风险和欲望对象。
```

L2-03 负责角色如何在这些限制中形成心理驱动：

```text
人格模型
欲望轴
执念
情绪迁移
关系位置
行为一致性
```

边界句：

```text
世界提供什么欲望对象和压力结构 → L2-04。
角色为什么抓住这个欲望对象行动 → L2-03。
```

### 5.4 与 L2-05 市场体验能力的边界

L2-04 负责设定本身是否具备：

```text
中心价值
世界差异
符号入口
可持续玩法
世界后果
```

L2-05 负责这些能力是否转化为：

```text
读者投入
留存
传播
付费
平台收益
```

边界句：

```text
设定是否有世界差异和玩法 → L2-04。
读者是否愿意为这个差异继续看 → L2-05。
```

### 5.5 与 L2-06 系统一致性与状态管理能力的边界

L2-04 负责规则如何设计。

L2-06 负责规则状态是否被保存、继承、版本一致。

边界句：

```text
规则如何成立 → L2-04。
规则是否前后一致、是否被旧文件污染 → L2-06。
```

### 5.6 与 L1.5 的派单边界

L2-04 发现越界问题时，不得直接调用其他 L2。

正确动作：

```text
输出“疑似越界 / 建议回 L1.5 重路由”
由 L1.5 决定是否派往 L2-01 / L2-02 / L2-03 / L2-05 / L2-06 / L3。
```

L2-04 可以给出建议目标，但不能越权派单。

## 6. 核心定义

### 6.1 创意不是新名词

创意设定不是给旧东西换名字。

创意设定是：

```text
在稳定边界内建立可运行模型，
通过压缩入口让读者进入，
通过有效偏离打破读者预期，
并围绕中心价值持续制造选择、冲突和后果。
```

### 6.2 设定不是说明书

设定成立不等于解释完整。

设定成立的最低要求是：

```text
它必须定义边界。
它必须建立入口。
它必须打破预期。
它必须围绕中心价值运行。
它必须制造压力。
它必须制造选择。
它必须制造代价。
它必须改变人物行为。
它必须改变事件可能性。
它必须能持续产生世界后果。
```

### 6.3 创意设定本体

```text
创意设定能力
=
中心价值
+
约束边界
+
压缩入口
+
有效偏离
+
可运行认知模型
+
世界运行后果
+
可持续玩法
```

### 6.4 一句话定义

```text
好设定 = 围绕中心价值运行的、可参与、可预测、可验证的认知模型。
```

## 7. 创意设定生成闭环

L2-04 的主运行链为：

```text
原始题材 / 设定问题包
↓
中心价值定位
↓
约束边界建立
↓
压缩入口建立
↓
有效偏离设计
↓
可运行认知模型
↓
世界运行后果
↓
可持续玩法
↓
创意设定体验成立
```

### 7.1 原始题材 / 设定问题包

输入对象：

```text
题材
世界观
规则
金手指
组织
职业
法则
资源
禁忌
地图
阶层
能力系统
```

L2-04 不直接把素材扩写成设定百科。

素材必须先被转译为“世界规则与差异问题”。

### 7.2 中心价值定位

回答：

```text
这个世界最想让读者关注什么价值？
所有规则、冲突、选择和代价围绕什么运行？
```

中心价值可以是：

```text
生存
成长
身份
权力
财富
自由
复仇
秩序
禁忌
知识
代价
交换
意义
```

失败风险：

```text
世界很大，但没有重心。
设定很多，但不能驱动冲突。
规则复杂，但不围绕同一个价值运行。
读者不知道该围绕什么下注。
```

### 7.3 约束边界建立

回答：

```text
什么能发生？
什么不能发生？
边界内有什么秩序与代价？
```

约束必须明确：

```text
限制对象
触发条件
允许行为
禁止行为
代价
收益
反噬
例外
漏洞
```

失败风险：

```text
世界观 / 规则不统一。
规则前后冲突。
设定自相矛盾。
边界不清。
```

### 7.4 压缩入口建立

回答：

```text
复杂世界如何被一个可记忆入口承载？
读者如何快速进入这个世界？
```

压缩入口可以是：

```text
一个词
一个物件
一个制度
一个禁忌
一个职业
一个动作
一个仪式
一个标记
一个空间
一个代价公式
```

失败风险：

```text
世界信息太散。
读者记不住。
传播时只能长篇解释。
缺少识别度与传播力。
```

### 7.5 有效偏离设计

回答：

```text
这个设定如何打破读者预期？
偏离是否在边界内成立？
偏离是否产生后果？
```

有效偏离必须满足：

```text
不是猎奇。
不是噱头。
不是随机反转。
必须改变读者对规则、选择、风险、价值或世界后果的判断。
```

失败风险：

```text
套路化。
缺乏新意。
读者一眼猜完。
难以产生惊喜与冲击。
```

### 7.6 可运行认知模型

回答：

```text
读者能否用一句“如果……就……”理解这个世界如何运行？
```

格式：

```text
如果【角色/世界】触发【规则】，
就会产生【代价/收益/状态变化】，
从而限制或打开【下一步选择】。
```

可运行模型必须具备：

```text
可预测
可参与
可验证
```

失败风险：

```text
世界无法运行。
规则无法预测。
读者无法理解因果。
设定只存在于说明，不进入行为。
```

### 7.7 世界运行后果

回答：

```text
如果这个规则存在，世界会变成什么样？
社会、组织、职业、资源、权力、人物关系会如何改变？
```

后果维度：

```text
资源分配
阶层结构
职业分化
组织制度
法律禁忌
交易方式
风险来源
社交关系
暴力形态
知识结构
```

失败风险：

```text
规则和世界脱节。
设定只影响主角，不影响社会。
世界没有自洽后果。
规则存在但生态不变。
```

### 7.8 可持续玩法

回答：

```text
这个设定能不能持续产生新问题、新冲突和新变化？
```

玩法来源：

```text
规则升级
代价扩大
漏洞暴露
组织反制
资源争夺
身份变化
信息解锁
禁忌试探
外部势力介入
```

失败风险：

```text
一章用完。
只够开头。
没有升级空间。
没有反制空间。
没有长期变化。
```

### 7.9 创意设定体验成立

创意设定体验成立的最低标准：

```text
读者能一句话说出这个世界的中心价值。
读者知道规则边界。
读者能通过入口记住这个世界。
读者知道偏离点在哪里。
读者能预测规则后果。
读者能参与角色选择。
读者相信这个世界会持续产生新后果。
```

## 8. L2-04 核心任务

L2-04 只做十三件事：

```text
1. 判断设定是否有中心价值
2. 判断规则 / 世界约束是否统一
3. 判断是否有压缩入口
4. 判断是否有有效偏离
5. 判断设定是否摆脱同质化
6. 判断规则是否制造选择
7. 判断规则是否制造代价
8. 判断设定是否有可运行认知模型
9. 判断设定是否有世界运行后果
10. 判断设定是否有可持续玩法
```

## 11. 主失败仲裁规则

当一个设定模块同时出现多个创意失败时，L2-04 必须先判断主失败。

```text
P1 中心价值缺失

P4 缺乏有效偏离

P11 可持续玩法不足

```

仲裁原则：

```text
先修中心价值，再修约束。
先修约束，再修压缩。
先修压缩，再修偏离。
先修可运行模型，再修符号。
先修世界后果，再修长期玩法。
输入不足时，不强行判断。
发现越界时，回 L1.5 重路由。
```

## 12. 创意设定失败类型

### F01 中心价值缺失

定义：

```text
世界没有足以统治全部规则、设定、冲突和选择的中心价值。
```

表现：

```text
世界很大，但没有重心。
设定很多，但不能驱动冲突。
规则复杂，但不围绕同一价值运行。
读者不知道该围绕什么下注。
```

修复规则：

```text
找到一个足以统治整个世界的中心价值。
所有规则、事件、设定必须服务于或冲突于这个价值。
```

验收标准：

```text
能否回答：
这个世界最值得争夺、害怕、交换、追求或违反的东西是什么？
```

---

### F02 约束不统一

定义：

```text
规则和设定没有稳定边界，导致世界观 / 规则不统一。
```

表现：

```text
设定自相矛盾。
规则前后冲突。
边界不清。
违反规则没有后果。
世界无法形成秩序。
```

修复规则：

```text
用约束定义边界。
明确什么能发生、什么不能发生、违反边界有什么代价。
```

验收标准：

```text
能否回答：
这个世界里，谁在什么条件下不能做什么？
```

---

### F03 缺乏有效压缩

定义：

```text
复杂世界没有被压缩成可记忆、可传播、可进入的认知入口。
```

表现：

```text
设定只能长篇解释。
读者记不住核心。
传播时无法复述。
缺少识别度。
没有核心物件、制度、标记、职业或禁忌。
```

修复规则：

```text
建立压缩入口。
把复杂世界压缩成一个词、物件、制度、动作、标记、职业、禁忌或代价公式。
```

验收标准：

```text
读者能否用一个入口记住这个世界？
```

---

### F04 缺乏有效偏离

定义：

```text
设定没有在边界内打破读者预期，无法产生新鲜感和创新体验。
```

表现：

```text
套路化。
缺乏新意。
读者一眼猜完。
反常识只是噱头。
偏离不产生后果。
```

修复规则：

```text
设计有效偏离。
偏离必须改变读者对规则、选择、风险、价值或世界后果的判断。
```

验收标准：

```text
读者是否会产生：
原来这个世界不是我以为的那样。
```

---

### F05 可运行认知模型缺失

定义：

```text
读者无法在脑中运行这个世界。
```

表现：

```text
规则太散。
设定靠解释维持。
无法预测触发后果。
世界运行缺少因果公式。
读者无法参与推演。
```

修复规则：

```text
设定必须压缩成“如果……就……”的运行模型。
```

验收标准：

```text
如果【角色/世界】触发【规则】，
就会产生【代价/收益/状态变化】，
从而限制或打开【下一步选择】。
```

---

### F06 世界 / 规则符号弱化

定义：

```text
设定没有可记忆、可复现、可传播的世界 / 规则符号。
```

表现：

```text
记不住。
传不开。
缺乏识别度。
缺少传播力。
读者无法用符号复述世界。
```

修复规则：

```text
建立世界 / 规则符号。
符号必须压缩中心价值、规则边界、代价、禁忌或世界后果。
```

验收标准：

```text
能否用一个词、物件、制度、动作、标记或画面让读者记住世界规则？
```

---

### F07 设定同质化

定义：

```text
设定没有改变同类题材的默认认知，只是换名、换壳、换术语。
```

表现：

```text
旧套路改名。
世界规则与同类作品无差异。
世界差异可被其他作品替换。
读者不用更新判断。
```

修复规则：

```text
设定必须制造明确差异源。
差异源必须改变读者对规则、选择、风险、价值或世界后果的判断。
```

验收标准：

```text
它与同类题材相比，改变了哪一个默认判断？
```

---

### F08 规则无法制造选择

定义：

```text
规则存在，但不迫使角色在多个有代价的方向之间选择。
```

表现：

```text
设定解释很多，行动选择不变。
规则不制造两难。
角色不用权衡。
冲突不贴脸。
```

修复规则：

```text
规则必须制造选择分叉。
每个分叉都必须有收益与代价。
```

验收标准：

```text
这个规则迫使角色在 A 与 B 之间怎么选？
```

---

### F09 规则没有代价

定义：

```text
角色使用规则、违反规则或绕过规则时没有代价。
```

表现：

```text
能力免费。
违反无惩罚。
收益无反噬。
选择后局面不变。
```

修复规则：

```text
每个关键规则必须绑定代价、风险、反噬或机会成本。
```

验收标准：

```text
得到这个收益，会失去什么？
```

---

### F10 世界运行后果缺失

定义：

```text
规则存在，但没有改变世界生态、社会结构、组织制度或人物关系。
```

表现：

```text
规则只影响主角。
社会不受规则影响。
组织制度没有变化。
资源分配不变。
世界像背景板。
```

修复规则：

```text
关键规则必须推出世界运行后果。
```

验收标准：

```text
如果这个规则存在，社会、组织、职业、资源和关系会变成什么样？
```

---

### F11 可持续玩法不足

定义：

```text
设定只够开头使用，无法持续制造新问题、新冲突和新变化。
```

表现：

```text
一章用完。
没有升级空间。
没有反制空间。
没有漏洞暴露。
没有组织介入。
没有长期变化。
```

修复规则：

```text
设定必须具备可持续玩法来源。
```

验收标准：

```text
这个设定能否持续产出：
升级、反制、漏洞、禁忌、资源争夺、身份变化、信息解锁？
```

---

### F12 验收不可判 / 疑似越界

定义：

```text
输入不足以判断创意设定失败，或主问题不属于创意设定层。
```

典型输入不足：

```text
没有设定文本
没有规则描述
没有题材范围
没有中心价值
没有失败位置
没有需要验收的设定单元
```

典型越界：

```text
主线发散 → L2-01
AI味 / 语言重复 → L2-02
人格欲望不足 → L2-03
留存付费弱 → L2-05
状态失忆 / 真源污染 → L2-06
执行协议错误 → L3
```

修复规则：

```text
L2-04 不硬判。
输出“需要补输入”或“建议回 L1.5 重路由”。
```

## 13. 修复动作库

L2-04 可输出以下修复动作：

```text
中心价值锁定：
明确世界最核心的争夺、恐惧、交换、追求或禁忌对象。

约束边界表：
定义什么能发生、什么不能发生、违反边界有什么代价。

压缩入口设计：
把复杂世界压缩成词、物件、制度、动作、标记、职业、禁忌或代价公式。

有效偏离设计：
在边界内打破读者预期，并绑定压力、收益、代价或后果。

运行模型压缩：
将设定压缩成“如果……就……”的可运行公式。

世界 / 规则符号建立：
建立能承载中心价值、规则边界、代价、禁忌或后果的符号。

规则压力表：
把规则转化成压力、选择、代价、风险和反噬。

选择分叉表：
列出角色被迫面对的 A/B 选择，以及各自收益与代价。

代价绑定：
给能力、资源、规则、收益绑定失去项、反噬项或机会成本。

世界后果推演：
推出规则对社会、组织、职业、资源、关系、暴力形态的影响。

可持续玩法表：
列出升级、反制、漏洞、禁忌、资源争夺、身份变化、信息解锁等长期玩法来源。

设定差异表：
对比同类题材，列出差异点、后果、风险和可持续性。

建议回 L1.5 重路由：
当主失败不是创意设定问题时，生成疑似越界说明、建议目标和回 L1.5 重路由要求。
```
