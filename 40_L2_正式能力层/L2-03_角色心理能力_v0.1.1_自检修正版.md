# L2-03 角色心理能力 v0.1.1 自检修正版

## 0. 文件定位

本文件属于 XC-UE 正式 L2 能力层的候选初版。

```text
40_L2_正式能力层/
└── L2-03_角色心理能力_v0.1.1_自检修正版.md
```

本文件只定义“角色心理能力”。

它不生成正文。
它不写章节。
它不写角色小传。
它不写人物设定百科。
它不把“性格标签”当作人格。
它不把“情绪描写”当作心理成立。
它不把“说话很多”当作对话自然。
它不替代 L1 闸门判断。
它不替代 L1.5 路由矩阵。
它不替代 L3 Cursor 执行协议。

本文件的目标是：

```text
把角色与心理问题从“人物假 / 动机弱 / 情绪假 / 对话假”
收束为可仲裁、可修复、可回流的人格驱动能力。
```

## 0.1 自检修正摘要

v0.1 方向基本正确，但存在五个边界风险：

```text
1. “动机弱”容易与 L2-01 的目标轴缺失混淆。
2. “对话模式化”容易与 L2-02 的语言表达失败混淆。
3. “情绪线不自然”容易与 L2-01 的读者心理映射混淆。
4. “能力”容易与 L2-04 的能力系统 / 世界规则混淆。
5. “真实角色”容易被误解为人物小传或设定百科，而不是可验收的人格驱动系统。
```

v0.1.1 的修正策略：

```text
动机弱：
目标轴缺失归 L2-01；欲望、信念、恐惧、过去导致的行动驱动不足归 L2-03。

对话模式化：
句子表达、节奏、语境锁定归 L2-02；隐藏目的、表达约束、潜台词动机归 L2-03。

情绪线不自然：
读者心理映射归 L2-01；角色内部心理迁移归 L2-03。

能力：
世界能力系统归 L2-04；角色可用资源、行动手段、限制与代价归 L2-03。

真实角色：
不写人物小传，只建立“欲望 + 信念 + 恐惧 + 能力/资源 + 过去”的人格驱动系统。
```

本版不扩写人物设定，只做边界加硬、术语校准与验收修正。

## 2. 能力定位

L2-03 是 XC-UE 的角色心理专题能力。

它回答的问题是：

```text
谁在行动？
为什么行动？
为什么这个人不是标签，而是有内部驱动、心理迁移和隐藏目的的动态存在？
```

L2-03 的核心职责：

```text
识别角色心理失败
↓
仲裁主失败
↓
输出角色心理修复作业
↓
回流 L1 / L1.5 验收
```

一句话定位：

```text
L2-03 负责构建能产生行为、情绪、语言和变化的人格驱动系统。
```

### 2.1 角色心理状态单元

L2-03 的最小处理对象不是“人物简介”，而是角色心理状态单元。

```text
角色心理状态单元
=
角色在某个场景、关系、压力或事件刺激下，
由欲望、信念、恐惧、能力/资源、过去共同生成的
行为倾向、情绪变化、语言策略与后续人格变化。
```

角色心理状态单元可以落在：

```text
单角色
单场景
单关系
单对话
单情绪线
多章角色线
```

禁止把角色心理状态单元写成人物百科。

## 3. L2-03 的根部结构

### 3.1 三大表象问题

角色与心理问题的表象不是“人物不够多”，而是：

```text
1. 人物千篇一律
2. 内心独白和情感线不自然
3. 对话模式化
```

对应表现：

```text
人物千篇一律：
性格模板化，缺少多维度，矛盾和成长线不足。

内心独白和情感线不自然：
情绪变化与事件不匹配，缺少真实感。

对话模式化：
通俗满但口语化不足，缺少潜台词、情绪波动和表达约束。
```

### 3.2 三大根本原因

表象背后的根本原因是：

```text
1. 人格被建模成标签
2. 只写情绪结果
3. 只存在信息层
```

解释：

```text
人格被建模成标签：
把人物写成“冷静、善良、腹黑、热血”等属性集合，缺少内部矛盾。

只写情绪结果：
直接写愤怒、痛苦、感动、害怕，不写心理迁移过程。

只存在信息层：
对话只是交换信息，没有目的、试探、压迫、遮掩和关系变化。
```

### 3.3 三大核心缺失

```text
缺少矛盾
缺少迁移
缺少隐藏目的
```

进一步解释：

```text
缺少矛盾：
没有冲突，就没有选择。
没有选择，就无法暴露真实人格。

缺少迁移：
没有前后状态变化，情绪无法真实形成。

缺少隐藏目的：
没有潜台词和表达约束，人物不像真人在交流。
```

## 4. 角色内部结构

真实人物的根本组成：

```text
欲望
+
信念
+
恐惧
+
能力
+
过去
=
人格驱动系统
```

### 4.1 欲望

回答：

```text
我想要什么？
```

功能：

```text
提供目标、需求、行动方向和主动性。
```

失败表现：

```text
角色不主动。
行动像被剧情推。
不知道角色为什么非要做这件事。
```

### 4.2 信念

回答：

```text
我相信什么？
```

功能：

```text
提供价值观、判断标准和选择原则。
```

失败表现：

```text
角色选择无原则。
今天这样想，明天那样想。
行为无法形成一致性。
```

### 4.3 恐惧

回答：

```text
我害怕什么？
```

功能：

```text
提供弱点、不安、回避、羞耻和保护机制。
```

失败表现：

```text
角色没有脆弱处。
冲突不能逼出真实反应。
角色没有被击中的地方。
```

### 4.4 能力

回答：

```text
我能做什么？
```

功能：

```text
提供资源、技能、手段、限制和行动方式。
```

失败表现：

```text
角色想做什么都能做。
手段和人格无关。
能力没有带来限制和代价。
```

### 4.5 过去

回答：

```text
我经历过什么？
```

功能：

```text
提供经历、创伤、习惯、记忆、债务、承诺和人格来源。
```

失败表现：

```text
角色像凭空出现。
经历不影响选择。
过去没有改变语言、行为和关系。
```

## 5. 三大生成系统

L2-03 有三条核心生成链：

```text
真实行为生成链
真实情感生成链
真实对话生成链
```

### 5.1 真实行为生成链

对应表象问题：

```text
人物千篇一律
```

真实行为不是“作者安排角色做事”。

真实行为链为：

```text
内部矛盾
↓
产生选择
↓
暴露人格
```

矛盾来源：

```text
欲望 vs 信念
欲望 vs 恐惧
信念 vs 现实
过去 vs 现在
能力 vs 目标
```

选择模式：

```text
不同选择 → 不同人格
相同行为，动机不同，人格完全不同
```

验收标准：

```text
角色当前行为是否由内部矛盾推出？
如果换一个人，是否会做出不同选择？
```

### 5.2 真实情感生成链

对应表象问题：

```text
内心独白和情感线不自然
```

真实情感不是情绪结果。

真实情感链为：

```text
事件刺激
↓
心理评估
↓
情绪反应
↓
行为表达
```

心理迁移链为：

```text
输入
↓
评估
↓
冲突
↓
偏移
↓
落差
↓
情绪
```

核心定义：

```text
情绪 = 心理位置变化。
```

补充定义：

```text
情绪不是直接告知。
情绪必须经过迁移过程，才有真实感。
```

验收标准：

```text
情绪是否有前后状态变化？
情绪是否能追溯到心理评估和内部冲突？
```

### 5.3 真实对话生成链

对应表象问题：

```text
对话模式化
```

真实对话不是信息交换。

真实对话链为：

```text
隐藏目的
↓
表达约束
↓
语言变形
↓
潜台词
```

隐藏目的类型：

```text
获取信息
施压
建立关系
试探
掩饰
测试底线
控制
情绪宣泄
引导对方
```

核心定义：

```text
同一句话，不同目的 → 完全不同含义。
```

验收标准：

```text
每句关键对话是否有隐藏目的？
对话是否受关系、场合、风险和表达约束影响？
```

## 6. 角色动态运行系统

角色不是静态设定。

角色动态运行链为：

```text
欲望驱动
↓
遇到阻力
↓
内部冲突
↓
做出选择
↓
产生后果
↓
更新信念
↓
改变人格
↓
影响语言与行为
```

解释：

```text
欲望驱动：
角色因目标和需求进入行动。

遇到阻力：
世界、他人、规则、资源或自身弱点制造阻碍。

内部冲突：
欲望、信念、恐惧、能力/资源、过去发生冲突。

做出选择：
选择暴露人格。

产生后果：
后果改变关系、状态、代价和下一步可能性。

更新信念：
角色对世界、自我、他人、规则的判断发生变化。

改变人格：
人格不是突变，而是被选择和后果慢慢压弯。

影响语言与行为：
变化必须进入后续行动、语气、沉默、对话、关系和选择。
```

## 7. 三层边界

角色心理能力必须放在小说三层系统中理解。

```text
世界层：写什么
叙事层：怎么发生
角色层：谁在行动
文风层：怎么说出来
```

对应能力层：

```text
世界层 → L2-04 创意设定能力
叙事层 → L2-01 叙事结构能力
角色层 → L2-03 角色心理能力
文风层 → L2-02 文风语言能力
```

L2-03 只处理“谁在行动、为什么行动、如何变化”。

## 7.1 边界加硬：角色心理问题的归属判定

### 7.1.1 动机弱归属

```text
目标轴缺失：
角色不知道当前要完成什么结构目标。
→ L2-01

内部驱动不足：
角色没有欲望、信念、恐惧、能力/资源、过去来支持行动。
→ L2-03
```

判定句：

```text
“他要做什么”不清楚，多数是 L2-01。
“他为什么非要这样做”不成立，多数是 L2-03。
```

### 7.1.2 情绪线归属

```text
读者如何感受、期待是否被接住：
→ L2-01 的心理映射

角色内部情绪如何从前一状态迁移到后一状态：
→ L2-03 的心理迁移
```

判定句：

```text
读者感受断了，查 L2-01。
角色情绪突然跳了，查 L2-03。
```

### 7.1.3 对话问题归属

```text
句子信息增量、语境锁定、语言节奏、AI味：
→ L2-02

隐藏目的、表达约束、潜台词、关系动作：
→ L2-03
```

判定句：

```text
这句话怎么说得更像现场语言，查 L2-02。
这个人为什么不明说、为什么这样试探，查 L2-03。
```

### 7.1.4 能力问题归属

```text
世界能力系统、规则、金手指机制：
→ L2-04

角色当前能用什么、不能用什么、代价是什么、因此如何选择：
→ L2-03

能力状态是否被后文继承：
→ L2-06
```

## 8. 与其他 L2 的边界

### 8.1 与 L2-01 叙事结构能力的边界

L2-01 负责目标轴、条件链、因果组织和事件结构。

L2-03 负责角色内部为什么会选择这条行动。

边界句：

```text
目标轴缺失导致的动机弱 → L2-01。
欲望、信念、恐惧、过去、心理迁移导致的动机弱 → L2-03。
```

### 8.2 与 L2-02 文风语言能力的边界

L2-02 负责语言表达是否有增量、语境锁定和语言节奏。

L2-03 负责对话背后的隐藏目的、心理位置和关系动机。

边界句：

```text
这句话怎么说得更有现场感 → L2-02。
这个人为什么只能这样说、为什么不明说 → L2-03。
```

### 8.3 与 L2-04 创意设定能力的边界

L2-04 负责世界提供什么欲望对象、限制和诱惑。

L2-03 负责角色如何把这些对象转化为欲望、恐惧、信念和选择。

边界句：

```text
世界提供什么压力结构 → L2-04。
角色为什么被这个压力击中 → L2-03。
```

### 8.4 与 L2-05 市场体验能力的边界

L2-03 负责角色是否真实、独特、可变化。

L2-05 负责这种角色是否带来读者投入、留存、传播和付费。

边界句：

```text
角色是否有血有肉 → L2-03。
读者是否愿意为这个角色继续看 → L2-05。
```

### 8.5 与 L2-06 系统一致性与状态管理能力的边界

L2-03 负责心理变化如何成立。

L2-06 负责角色状态是否被长期保存和继承。

边界句：

```text
角色为何改变 → L2-03。
角色改变后是否被后文继承 → L2-06。
```

### 8.6 与 L1.5 的派单边界

L2-03 发现越界问题时，不得直接调用其他 L2。

正确动作：

```text
输出“疑似越界 / 建议回 L1.5 重路由”
由 L1.5 决定是否派往 L2-01 / L2-02 / L2-04 / L2-05 / L2-06 / L3。
```

L2-03 可以给出建议目标，但不能越权派单。

## 9. 输入边界

### 9.1 允许输入

L2-03 接收来自 L1 / L1.5 的角色心理问题包。

标准输入格式：

```yaml
failure_type: 角色心理失败
failure_signal:
  - 人物模板化
  - 动机弱
  - 情绪假
  - 对话假
  - 角色被剧情推着走
  - 缺少欲望
  - 缺少信念
  - 缺少恐惧
  - 缺少过去
  - 缺少心理迁移
  - 缺少隐藏目的
  - 角色长期行为一致性失败
  - 表达约束缺失
  - 潜台词缺失
severity: low | medium | high
source_layer: L1 | L1.5
route_target: L2-03
context_scope: 单角色 | 单场景 | 单关系 | 单对话 | 单情绪线 | 多章角色线
```

### 9.2 禁止输入

L2-03 禁止直接接收：

```text
“帮我写一个人设”
“帮我写人物小传”
“让角色更立体”
“让他更疯一点”
“让她更有魅力”
“写一段心理描写”
“写几句对话”
“直接改正文”
```

如需处理，必须先由 L1 / L1.5 抽象成角色心理失败包。

## 11. 主失败仲裁规则

当一个角色问题同时出现多个失败时，L2-03 必须先判断主失败。

主失败优先级如下：

```text
P1 人格驱动系统缺失
P2 欲望缺失
P3 信念缺失
P4 恐惧 / 弱点缺失
P5 能力/资源与限制脱节
P6 过去不影响现在
P7 内部矛盾缺失
P8 心理迁移缺失
P9 隐藏目的缺失
P10 对话表达约束缺失
P11 角色动态更新失败
P12 验收不可判 / 疑似越界
```

仲裁原则：

```text
先修人格驱动，再修行为。
先修欲望、信念、恐惧，再修情绪。
先修心理迁移，再修情绪结果。
先修隐藏目的，再修对话表层。
对话表层语言问题不在 L2-03 内硬修，必须建议回 L1.5 转 L2-02。
先修动态更新，再修长期角色线。
角色状态继承问题不在 L2-03 内硬修，必须建议回 L1.5 转 L2-06。
输入不足时，不强行判断。
发现越界时，回 L1.5 重路由。
```

## 12. 角色心理失败类型

### F01 人格驱动系统缺失

定义：

```text
角色缺少欲望、信念、恐惧、能力/资源、过去构成的内部驱动系统。
```

表现：

```text
人物只是标签。
行为像剧情安排。
情绪像作者标注。
对话像信息输出。
角色没有独特反应模式。
```

修复规则：

```text
必须建立人格驱动系统：
欲望 + 信念 + 恐惧 + 能力/资源 + 过去。
但不得写成人物小传或标签集合。
```

验收标准：

```text
能否回答：
这个人想要什么、相信什么、害怕什么、能做什么、经历过什么？
```

---

### F02 欲望缺失

定义：

```text
角色没有明确目标、需求或主动追求。
```

表现：

```text
角色不主动。
行动被剧情推。
遇事只是反应。
不知道他为什么非要做这件事。
```

修复规则：

```text
给角色绑定欲望对象。
欲望必须能驱动行动，并在遇到阻力时产生冲突。
```

验收标准：

```text
角色当前最想要什么？
失去它会怎样？
```

---

### F03 信念缺失

定义：

```text
角色没有稳定判断原则和价值观。
```

表现：

```text
选择没有原则。
行为前后不一致。
角色为剧情需要随时改想法。
```

修复规则：

```text
给角色建立信念。
信念必须能影响选择，并在冲突中受到挑战。
```

验收标准：

```text
这个人相信什么？
他会为了这个信念拒绝什么？
```

---

### F04 恐惧 / 弱点缺失

定义：

```text
角色没有真正害怕、回避、羞耻、不安或无法承受的东西。
```

表现：

```text
人物没有脆弱点。
冲突打不到角色。
角色不会被迫暴露真实反应。
```

修复规则：

```text
为角色绑定恐惧与弱点。
恐惧必须能在关键冲突中影响选择。
```

验收标准：

```text
这个人最怕什么发生？
这个恐惧如何改变他的行动？
```

---

### F05 能力/资源与限制脱节

定义：

```text
角色能力/资源只作为工具存在，没有限制、代价和行为风格。
```

表现：

```text
想做什么都能做。
能力和人格无关。
能力/资源没有制造局限。
手段没有带出人物特点。
```

修复规则：

```text
能力/资源必须绑定限制、代价、习惯、偏好和人格风格。
```

验收标准：

```text
这个人能做什么？
不能做什么？
他会用什么方式解决问题？
```

---

### F06 过去不影响现在

定义：

```text
角色过去没有持续影响当前欲望、信念、恐惧、语言和选择。
```

表现：

```text
角色像凭空出现。
经历只是背景说明。
创伤、债务、承诺、失败不影响现在。
```

修复规则：

```text
过去必须转化为当前限制、习惯、恐惧、信念、债务或关系压力。
```

验收标准：

```text
如果删除这段过去，角色现在的选择是否会变化？
```

---

### F07 内部矛盾缺失

定义：

```text
角色内部没有冲突，因此无法通过选择暴露人格。
```

表现：

```text
人物想法过于平直。
没有两难。
没有选择代价。
没有真实人格暴露点。
```

修复规则：

```text
必须建立内部矛盾。
矛盾可以来自欲望、信念、恐惧、能力/资源、过去之间的冲突。
```

验收标准：

```text
角色当前被哪两个内部力量拉扯？
他选择哪边，暴露了什么人格？
```

---

### F08 心理迁移缺失

定义：

```text
角色内部情绪没有经过心理位置变化，只写了情绪结果。这里处理角色心理迁移，不处理读者心理映射。
```

表现：

```text
突然愤怒。
突然感动。
突然崩溃。
情绪与事件不匹配。
读者看不到心理变化过程。
```

修复规则：

```text
情绪必须经过：
输入 → 评估 → 冲突 → 偏移 → 落差 → 情绪。
```

验收标准：

```text
情绪是否能追溯到心理评估和状态变化？
```

---

### F09 隐藏目的缺失

定义：

```text
对话没有隐藏目的，只是传递信息。这里处理对话背后的心理目的，不处理句子润色。
```

表现：

```text
人物说话像旁白。
对话只负责解释设定。
没有试探、施压、掩饰、控制、引导或关系动作。
```

修复规则：

```text
每句关键对话必须绑定隐藏目的。
```

验收标准：

```text
这句话表面在说什么？
实际想得到什么？
不敢明说什么？
```

---

### F10 对话表达约束缺失

定义：

```text
对话没有受到身份、关系、场合、风险、情绪和隐藏目的约束。
```

表现：

```text
谁都能这样说。
所有角色说话相似。
对话没有潜台词。
不受权力关系影响。
不受危险场合影响。
```

修复规则：

```text
对话必须受表达约束变形。
表达约束包括身份、关系、场合、风险、情绪、目的。
若问题主要是句式、节奏、AI味或信息增量，应回 L1.5 建议转 L2-02。
```

验收标准：

```text
换一个人、换一个关系、换一个场合，这句话是否必须改变？
```

---

### F11 角色动态更新失败

定义：

```text
角色选择和后果没有更新信念、人格、语言与行为。
```

表现：

```text
经历大事后像没变。
选择没有心理后果。
关系变化不影响说话。
成长线断裂。
```

修复规则：

```text
角色动态链必须闭合：
欲望驱动 → 遇到阻力 → 内部冲突 → 做出选择 → 产生后果 → 更新信念 → 改变人格 → 影响语言与行为。
```

验收标准：

```text
角色经历后，后续语言、行为、选择是否发生可追踪变化？
```

---

### F12 验收不可判 / 疑似越界

定义：

```text
输入不足以判断角色心理失败，或主问题不属于 L2-03。
```

典型输入不足：

```text
没有角色当前目标
没有事件刺激
没有关系对象
没有角色过去
没有对话场景
没有失败位置
没有当前心理状态
```

典型越界：

```text
主线发散 → L2-01
AI味 / 语言重复 → L2-02
世界规则空洞 → L2-04
留存付费弱 → L2-05
状态失忆 / 真源污染 → L2-06
执行协议错误 → L3
```

修复规则：

```text
L2-03 不硬判。
输出“需要补输入”或“建议回 L1.5 重路由”。
```

## 13. 修复动作库

L2-03 可输出以下修复动作：

```text
人格驱动表：
建立欲望、信念、恐惧、能力/资源、过去五项结构。

欲望对象锁定：
明确角色最想获得、逃离、保护、证明或摧毁的东西。

信念句建立：
明确角色相信什么，以及他会为此拒绝什么。

恐惧 / 弱点绑定：
明确角色最怕什么被触发，以及触发后如何变形。

能力/资源限制表：
记录角色能做什么、拥有什么资源、不能做什么、做了有什么代价。

过去影响表：
把经历转化为当前限制、习惯、信念、恐惧、债务或关系压力。

内部矛盾表：
列出欲望、信念、恐惧、能力/资源、过去之间的冲突。

心理迁移链：
标注输入、评估、冲突、偏移、落差、情绪。

隐藏目的表：
标注关键对话的真实目的、遮掩内容和期望结果。

表达约束表：
标注身份、关系、场合、风险、情绪、目的如何限制说法。

角色动态更新表：
记录选择、后果、信念更新、人格变化、语言行为变化。

建议回 L1.5 重路由：
当主失败不是角色心理问题时，生成疑似越界说明、建议目标和回 L1.5 重路由要求。
```

## 14. 回流验收

L2-03 修复后，必须回流到 L1 / L1.5 验收。

常见回流位置：

```text
L1-01：内部创作成立闸门
L1.5：路由复核
```

验收问题：

```text
1. 角色是否有欲望？
2. 角色是否有信念？
3. 角色是否有恐惧或弱点？
4. 角色能力/资源是否带有限制和代价？
5. 角色过去是否影响现在？
6. 角色是否存在内部矛盾？
7. 行为是否由内部矛盾推出？
8. 情绪是否经过角色内部心理迁移？
9. 对话是否有隐藏目的？
10. 对话是否受表达约束影响？
11. 角色选择是否产生后果？
12. 后果是否更新信念、人格、语言和行为？
13. 是否形成有血有肉、可变化、独特的人格驱动系统？
14. 是否存在疑似越界、需要回 L1.5 重路由的问题？
```
