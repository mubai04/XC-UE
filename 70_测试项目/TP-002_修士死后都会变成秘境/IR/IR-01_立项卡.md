# IR-01 立项卡

> 来源：立项卡 v0.1  
> IR状态：当前  
> 用途：给 L1-01 / L1-02 / L1-03 / L2-05 提供项目级输入。

## 1. 项目基本信息

**项目名：** 《修士死后，都会变成秘境》

备选名：

- 《身死成境》
- 《我在仙门替死人开门》
- 《收尸十年，我打开了祖师秘境》

**目标平台：** 番茄小说，男频长篇  

**类型：** 东方仙侠＋秘境探索＋宗门悬疑＋成长升级  

**目标读者：** 喜欢修仙、秘境寻宝、宗门黑幕、捡漏成长和世界真相的男频读者。  

**主情绪：**

- 前期：捡漏、紧张、窥见秘密
- 中期：反杀、争夺、揭穿宗门黑幕
- 后期：壮阔、悲凉、重建修仙秩序

**核心阅读体验：** 进入一名死去修士留下的世界，看见他隐藏了一生的力量、秘密和执念。

## 2. 一句话卖点

**修士死后，毕生修为、记忆与执念都会化成秘境；主角的工作，是替死人打开秘境。**

## 3. 核心钩子

**读者为什么点开：** 标题给出反常识——修士不是留下尸体，而是变成秘境。熟悉元素（修士、秘境、传承）但关系变化，理解成本低。

**读者为什么看完第一章：** 一个没有修为的药园老人死后，竟然化成一座远超其身份的大型秘境，说明老人身份是假的。

**读者为什么追到第二章：** 老人遗言「第三扇门不能开」，但宗门长老进入后的第一道命令却是寻找第三扇门。

## 4. 最小爽感单位

本书稳定爽点：

- 别人只看见秘境宝物，主角能看见秘境为什么形成
- 宗门强者抢夺传承，主角提前知道秘境规则
- 被当作废物的死者，死后暴露真正身份
- 强者生前压住的秘密，在死后世界中重新出现
- 主角从替死人开门，成长为决定世界如何留下遗产的人

## 5. 长期玩法

100 章后还能继续玩的原因：

- 一个遗境对应一个人物秘密
- 每次继承传承也继承未完成事务
- 第三扇门贯穿第一卷
- 「世界是否也是遗境」作为远期悬念

## 6. 风险预判

| 风险 | 说明 |
|---|---|
| 入口弱 | 第一章先讲制度历史会导致弃读 |
| 主角目标弱 | 若被动跌入秘境，驱动力不足 |
| 设定太重 | 旁白解释过多 |
| 秘境副本化 | 固定闯三关结构会丧失「死者一生化为世界」独特性 |
| 遗言等于答案 | 遗言只能给线索，不能直接给凶手/出口/传承位置 |
| 结构重复 | 每座秘境须由死者职业、人生、执念决定结构 |

## 7. 发布判断

**当前发布状态：** 可以开始制作第一章，不可直接发布。

**发布前阻断项：**

1. 第一章完整正文尚未完成
2. 药园老人三扇门分别代表什么尚未锁定
3. 第一卷最终收益尚未确定
4. 韩照骨为什么必须打开第三扇门尚未形成唯一答案
5. 主角第一项正式传承尚未确定
6. 需验证第一章是否在前 1000 字内让老人死亡并开始成境

**进入正文前必须锁定的三个答案：**

1. 第三扇门后究竟是什么
2. 周不问为什么选择陆沉舟
3. 陆沉舟第一卷得到什么永久能力，同时继承什么永久麻烦
