# RT-00 运行时索引

> 状态：测试组件 / 非 L4 / 不自动推演  
> 用途：规定 runtime 五张记录表如何与 IR 回写。  
> 边界：runtime 只记录状态变化，不自动决定剧情，不升正式 L4。

---

## 1. Runtime 当前定位

runtime 不是 L4。  
runtime 不是自动推演器。  
runtime 不是正文生成器。  
runtime 不是价值函数正式系统层。

runtime 当前只做：

```text
记录事件刺激
记录价值函数变化
记录角色状态迁移
记录目标变化
记录选择结果
标记需要回写的 IR 位置
```

---

## 2. Runtime 文件组

```text
runtime/
├── RT-00_运行时索引.md
├── RT-01_价值函数演化表.md
├── RT-02_角色状态迁移表.md
├── RT-03_事件刺激记录.md
├── RT-04_目标变化记录.md
└── RT-05_选择结果记录.md
```

---

## 3. Runtime → IR 回写规则

| Runtime 文件 | 记录内容 | 回写目标 |
|---|---|---|
| RT-01 价值函数演化表 | 价值权重变化 | IR-08 当前角色状态 / IR-03 角色动机表 |
| RT-02 角色状态迁移表 | 目标、情绪、资源、关系变化 | IR-08 状态快照 |
| RT-03 事件刺激记录 | 事件刺激与影响 | IR-04 事件链 |
| RT-04 目标变化记录 | 原目标 → 新目标 | IR-03 角色动机表 / IR-05 章节目标表 |
| RT-05 选择结果记录 | 删除可能性 / 新增约束 | IR-04 路径剪枝记录 / IR-02 禁止回潮 |

```text
事件刺激

```

---

## 6. 阻断条件

出现以下情况必须阻断：

1. Runtime 结果试图直接改正式正文。
2. Runtime 结果试图覆盖 L2 能力定义。
3. Runtime 自动推演下一章。
4. Runtime 回写 IR 但未记录来源事件。
5. Runtime 与正文事实冲突但未触发 L2-06。
